## 使用方法：
1. 雙擊bot.py啟動機器人（必須先安裝Python）
2. 新增一個名為TOKEN.txt的文件，裡面輸入機器人的TOKEN
3. 修改伺服器狀態的頻道ID

## 注意事項：
1. 伺服器關閉後，請將原本的狀態訊息刪除，以免造成頻道混亂
2. 如果不知道佔位符在寫什麼請不要隨意更改！
3. 記得安裝：requirements.txt
安裝教學：
```pip install -r requirements.txt```
